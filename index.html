Installation Instructions

Alpine Linux

Add our repository to your Alpine Linux system and install X11Libre:

1. Enable community repository (required for dependencies)

sed -i '/^#.*community/s/^#//' /etc/apk/repositories

Or manually uncomment the line containing "community" in /etc/apk/repositories

2. Add the repository key

wget -O /etc/apk/keys/builduser-68fcd79c.rsa.pub https://xlibre-alpine.github.io/website/main/builduser-68fcd79c.rsa.pub

Replace builduser-68fcd79c.rsa.pub with user-694af499.rsa.pub for ARM64 architecture.

3. Add the repository to your sources

echo "https://xlibre-alpine.github.io/website/main" >> /etc/apk/repositories

4. Update package index

apk update

5. Install X11Libre

apk add xlibre-xserver

Chimera Linux

Add our repository to your Chimera Linux system and install X11Libre:

1. Add the repository key

wget -O /etc/apk/keys/builduser-68fcd79c.rsa.pub https://xlibre-alpine.github.io/website/main/builduser-68fcd79c.rsa.pub

Replace builduser-68fcd79c.rsa.pub with user-694af499.rsa.pub for ARM64 architecture.

2. Add the repository to your sources

echo "https://xlibre-alpine.github.io/website/chimera/main" >> /etc/apk/repositories

3. Update package index

apk update

4. Install X11Libre

apk add xserver-xlibre
